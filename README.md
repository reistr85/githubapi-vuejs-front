# github-front

## Get started
```
1-git clone https://github.com/reistr85/githubapi-vuejs-front.git;
2-cd githubapi-vuejs-front;
3-create the .env file in the project root with the content: VUE_APP_URL_API=http://localhost:9001/api/v1;
4-yarn install;
5-yarn serve (for development);
6-yarn build (for production);
```
